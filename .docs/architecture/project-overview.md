# 프로젝트 개요

## 프로젝트명

Harness Lab

## 한 줄 소개

컨텍스트 엔지니어링과 하네스 엔지니어링을 동일한 작업 조건에서 비교하고, 그 차이를 구조화된 데이터와 결과로 관리하기 위한 프로젝트

## 프로젝트 정의

이 프로젝트는 컨텍스트 엔지니어링과 하네스 엔지니어링을 비교하기 위해 필요한 비교 기준, 실험 단위, 결과 기록 구조를 정의하고 이를 시스템으로 표현하는 것을 목표로 한다.

비교의 대상은 단순한 개념 설명이 아니라, 동일한 작업 조건에서 어떤 방식이 더 재현 가능하고, 검증 가능하며, 설명 가능한 결과를 만드는지에 있다.

이 문서는 프로젝트의 배경 서사보다 설계 범위와 구조 정의를 우선한다.

## 핵심 목표

- 컨텍스트 엔지니어링과 하네스 엔지니어링의 차이를 프로젝트 내부 기준으로 정의한다
- 동일한 시나리오에서 두 접근 방식을 비교할 수 있는 구조를 만든다
- 비교 결과를 점수, 메모, 근거 데이터 형태로 기록할 수 있게 한다
- 비교 결과를 시스템 안에서 설명 가능한 형태로 조회할 수 있게 한다

## 프로젝트 범위

이 프로젝트는 다음 범위를 핵심 범위로 둔다.

- 비교 대상 정의
- 비교 기준 정의
- 실험 시나리오 등록
- 실험 결과 기록
- 결과 비교 조회
